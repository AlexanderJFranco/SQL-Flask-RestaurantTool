CIS 4930 Project


What is the goal of this project?

The goal of this project is to send scheduled reminders to its users utilizing Twilio's SMS API for Python

What algorithm does this project use?

This application takes user input and creates a time object (in the form of YYYY/MM/DD/hh)
and compares it to a current time object that is created at run time.

How can this project be improved?

This project would be greatly improved if utilized with a premium Twilio accounnt, as reminders could be relayed through any
platform. Additionally, the program would work best in a system where it can be left constantly running on loop (not feasible on testing devices).
